대항해시대 온라인(파파야 한국 서버) 다중 계정 런처. 여러 계정을 한 창에서 관리·실행·재접속.

---

## 주요 기능

- **다중 계정 관리** — 최대 9개 계정을 카드 그리드로 정렬·편집·실행
- **1-클릭 재접속** — 끊긴 클라이언트를 카드 클릭 한 번으로 즉시 재접속; preflight 토큰 검증으로 만료 사전 감지
- **위치·해상도 프리셋** — 계정별 창 위치와 해상도를 각각 저장
- **USB 포터블** — ZIP 풀기만 하면 즉시 사용; 폴더 복사로 다른 PC 이동 가능
- **자동 업데이트** — 설치본은 앱 내 배지·모달로 새 버전 알림

### Windows SmartScreen 경고

코드사이닝 인증서 없이 배포되므로 첫 실행 시 경고가 표시됩니다.  
**추가 정보 → 실행** 순서로 클릭하면 정상 실행됩니다.  
릴리즈 바이너리는 minisign 서명이 함께 배포되어 변조 여부를 검증할 수 있습니다.

## 안전성

- **파파야 런처 작동 방식 기반** — 게임 바이너리 변조나 실행 중인 프로세스 메모리 접근은 하지 않습니다.
- **AES-256-GCM 로컬 암호화** — 계정 자격증명은 로컬에만 저장. 외부 서버 전송 없음, 텔레메트리 없음.
