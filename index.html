— 기능

계정 관리에 필요한 것만.

다중 계정 관리

계정을 카드 그리드로 정렬·편집·실행. 드래그로 순서 바꾸고 편집 다이얼로그에서 즉시 수정.

1-클릭 재접속

끊긴 클라이언트도 카드 한 번 클릭으로 즉시 재접속. preflight 토큰 검증으로 만료 사전 감지.

위치·해상도 프리셋

모니터 축소도에서 창 위치를 드래그로 지정. 해상도는 게임 내장 프리셋 그대로 선택.

USB 포터블

ZIP 풀기만 하면 바로 사용. 계정·설정은 폴더 내부 data/ 에 저장, %APPDATA% 건드리지 않고 폴더 통째로 다른 PC 이동.

커스텀 경로 지원

비표준 폴더에 게임을 설치했어도 설정 다이얼로그의 찾아보기 로 GVOnline.exe 직접 지정.

— 안전성

게임 프로세스는 건드리지 않는다.

→
파파야 런처 작동 방식 기반 게임 바이너리 변조나 실행 중인 프로세스 메모리 접근은 하지 않습니다.
→
AES-256-GCM 로컬 암호화 계정 자격증명은 SQLite + AES-256-GCM 으로 로컬에만 저장. 외부 서버 전송 없음, 텔레메트리 없음.

— 설치

1분이면 첫 실행.

1

다운로드

설치본 (권장). 시작 메뉴 등록, Program Files 설치, 자동 업데이트 지원.

Portable ZIP. 압축 풀어 아무데나 두고 실행. USB 이동 가능. 자동 업데이트 없음.

2

SmartScreen 우회

코드사이닝 인증서 없이 배포되므로 첫 실행 시 Windows SmartScreen 경고가 뜹니다.

추가 정보 → 실행 순서로 클릭하면 정상 실행됩니다. 릴리즈 바이너리는 minisign 으로 서명되어 변조 여부를 확인할 수 있습니다.

3

첫 실행

  • 게임 경로 자동 감지 확인 (상단 헤더)
  • 첫 계정 추가 — + 카드 클릭 후 파파야 아이디·비번 입력
  • 로그인 보안 확인 (Cloudflare) 통과 후 캐릭터 선택 화면 진입

— FAQ

자주 묻는 질문.

SmartScreen 경고 — 안전한가요?

코드사이닝 인증서가 없어 Windows 가 "알 수 없는 게시자"로 표시합니다. 우회하면 정상 동작. 릴리즈 바이너리는 minisign 서명이 함께 배포되어 변조 여부를 검증할 수 있습니다.

여러 PC 에서 같은 계정 관리 하려면?

Portable ZIP 을 USB 에 담아 이동하세요. data/ 폴더가 계정 정보를 보관합니다.

토큰이 자꾸 만료돼요

파파야 토큰 TTL 은 약 1시간. 만료 시 카드의 "다시 로그인" 버튼으로 재접속하세요. 앱 기동 시 preflight 로 자동 유효성 검사가 돌아가므로 대부분은 클릭 한 번으로 끝납니다.

업데이트는 자동인가요?

설치본은 자동 — 새 버전이 있으면 앱 내 배지·모달로 알립니다. Portable ZIP 은 수동 — 이 페이지에서 최신 ZIP 받아 폴더 교체.

데이터 초기화 · 로그 위치

설치본: %APPDATA%/dev.dho.manager/. Portable: data/. 폴더 삭제하면 초기화, logs/ 하위는 진단 로그.

GameGuard 에 안전한가요?

파파야 런처 작동 방식을 기반으로 동작합니다. 게임 프로세스 메모리나 바이너리를 변조하지 않습니다.

동시 몇 개까지 실행?

최대 2 계정 동시 (게임사 공식 권장치, 하드코딩).

v1 에서 업그레이드?

v2 는 클린 슬레이트입니다. 계정은 수동 재등록하세요 (보안 개선: AES-256-GCM 암호화).
